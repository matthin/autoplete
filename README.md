# Autoplete
Documentation can be found
[here](https://docs.matthin.com/autoplete/current/index.html).

# Developers
Requirements:
* Ruby v1.9.3+
* Bundler (`$ gem install bundler`)
* Node v0.10+

### Installing
```bash
$ bundle install
$ npm install
```

###Compiling
```bash
$ bundle exec rake compile
```

Please try to conform to a max line width of 80.
